# Financial Planner with Monte Carlo Simulation

## Overview

This project is a financial planning tool designed to forecast portfolio performance and assist with retirement planning. This project was completed as part of the Northwestern FinTech Bootcamp.

## Key Features

* **Monte Carlo Simulation:** Employs Monte Carlo forecasting ("MCForecastTools") to simulate thousands of potential outcomes for an investment portfolio.
* **Portfolio Analysis:** Calculates potential returns, risks, and the probability of achieving financial goals based on given inputs.
* **Retirement Planning:** Helps users assess the viability of their retirement savings strategy over the long term.

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* APIs (for pulling financial data)
* Matplotlib (for visualization)
